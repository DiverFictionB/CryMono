Description

CryMono brings the power of .NET into the world of CryENGINE3, allowing for game logic to be scripted in a much faster and more intuitive way than before.

Documentation

For instructions on all things CryMono, see our Confluence setup at docs.inkdev.net.

Need help?

If you need help with something the documentation doesn't cover, you can get help at the CryMono boards at CryDev.net, or our IRC channel (#CryMono) at the GameSurge network.
