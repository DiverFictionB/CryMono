Description

CryMono brings the power of .NET into the world of CryENGINE, allowing for game logic to be scripted in a much faster and more intuitive way than before.

Integrating into the GameDLL

  1. Include MonoDll/Headers/IMonoScriptSystem.h and MonoDll/Headers/CryMonoInitializationHelpers.h in GameStartup.cpp
  2. Add 'IMonoScriptSystem *IMonoScriptSystem::g_pThis = 0;' to GameStartup.cpp
  3. Add the line 'InitCryMono(pSystem, m_pFramework);' after the 'ModuleInitISystem(m_pFramework->GetISystem(),"CryGame");' line
  4. (Optional) Include MonoDll/Headers/IMonoScriptSystem.h in Game.cpp and add 'IMonoScriptSystem::g_pThis->RegisterFlownodes();' in CGame::RegisterGameFlowNodes
